# 2D Platformer (Unity)

## Overview
A 2D platformer game developed in Unity where the player navigates platforms, avoids obstacles, and interacts with the environment using physics-based movement.

## Gameplay Features
- Character Selection and Locking/Unlocking
- Level Selection and Locking/Unlocking
- Player movement using Rigidbody2D
- Jump mechanics with ground detection
- Collision handling using triggers and colliders
- Camera follow system
- Meely/Ranged enemy and obstacle

## Concepts & Systems Used
- Unity 2D Physics (Rigidbody2D, Collider2D)
- Update vs FixedUpdate
- Input handling
- Raycast / ground check logic
- Animator Controller & transitions
- Coroutines for timing

## Tools & Technologies
- Unity Engine
- C#
- Visual Studio

## Learning Outcomes
- Implemented physics-based 2D movement
- Understood Unity script lifecycle
- Practiced clean scripting and scene organization
- Improved debugging and problem-solving skills
